# Frequently Asked Questions

## 1. Can I customize the colors and fonts to match my clinic's brand?

Yes. All colors, fonts, and key spacing values are defined as CSS custom properties in a single stylesheet. You can update them in one place and the changes apply across all 16 pages. The included documentation explains each variable.

## 2. Do I need Webflow to use this template?

No. The template is built with pure HTML, CSS, and JavaScript. You can open and edit it with any code editor and deploy it to any hosting provider — Netlify, Vercel, shared hosting, or any platform that serves static files. However, CMS collection schemas are documented if you want to migrate the structure into Webflow.

## 3. Is this a Webflow template or cloneable project?

It is not a native Webflow project. It is a standalone HTML/CSS/JS codebase. The documentation includes 7 CMS collection schemas so a Webflow developer can recreate the data structure in Webflow if desired.

## 4. What about the images? Can I use the ones included?

The template includes AVIF placeholder images to preserve the layout and visual structure. These are placeholders only — you should replace them with your own clinic photography before going live. No stock photo licenses are included.

## 5. Do I need coding experience to customize this?

Basic familiarity with HTML and CSS is recommended. The code is clean and well-commented, and the documentation covers common changes step by step. For more advanced modifications, any front-end developer will be able to work with the codebase without difficulty.

## 6. How many projects can I use this for?

The standard license covers one commercial project — either your own clinic website or a single client project. If you are an agency that needs to use the template for multiple clients, an extended license is available. See the license file for details.

## 7. Can I resell or redistribute this template?

No. The license does not permit reselling, sublicensing, or redistributing the template files in any form. You may use it to build a website, but you may not sell the template itself.

## 8. Will I receive updates?

Bug fixes and minor improvements will be provided for six months from the date of purchase. You will be notified of any updates through the platform where you purchased the template.

## 9. What kind of support do you provide?

Support covers questions about the template's existing features, setup guidance, and documented customization. It does not cover custom development, content writing, or third-party integrations. See the support policy for full details and response times.

## 10. Is the template mobile-friendly?

Yes. The template is fully responsive with tested breakpoints at 1024px (tablet) and 680px (mobile). Navigation, layouts, images, and typography all adapt cleanly to smaller screens.
