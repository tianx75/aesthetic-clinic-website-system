# Aesthetic Clinic Website System

## Overview

A professionally designed, production-ready website system for medical aesthetics clinics. Sixteen fully built pages covering every aspect of a modern clinic's online presence — from treatment catalogs and technology showcases to consultation booking and clinical education.

Built with pure HTML, CSS, and JavaScript. No frameworks. No dependencies. No build tools required. Open the files in any editor, customize to your brand, and deploy to any hosting platform.

## What's Included

- 16 complete HTML pages, fully linked and styled
- Global CSS architecture using custom properties for easy theming
- Vanilla JavaScript for scroll animations and interactive components
- AVIF placeholder images sized and positioned for every section
- SVG favicon and Open Graph image
- 7 CMS collection schemas documented for Webflow or headless CMS migration
- 11 documentation files covering setup, customization, CMS structure, and deployment

## Key Features

- **Refined visual identity.** Cormorant Garamond and Montserrat typography paired with a cream, ivory, charcoal, and gold color palette. Every page maintains a consistent, premium aesthetic.
- **Filterable treatment catalog.** Clients can browse and filter treatments by category without page reloads.
- **Scroll-triggered animations.** Smooth entrance animations powered by IntersectionObserver — no animation libraries needed.
- **Fully responsive.** Tested breakpoints at 1024px and 680px. Clean layouts on desktop, tablet, and mobile.
- **Theming via CSS custom properties.** Change colors, fonts, and spacing from a single file. No search-and-replace across dozens of pages.
- **Zero dependencies.** No npm, no build step, no framework lock-in. Works with any hosting provider or site builder that accepts HTML.

## Customization

All visual theming is controlled through CSS custom properties defined in a single stylesheet. To match your clinic's brand:

1. Update the color variables (primary, secondary, accent, background, text)
2. Swap the font imports if you prefer different typefaces
3. Replace placeholder images with your own photography
4. Edit the HTML content to reflect your treatments, team, and clinic details

The included documentation walks through each step.
