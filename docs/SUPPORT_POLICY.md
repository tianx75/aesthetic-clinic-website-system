# Support Policy

## What Is Included

### Documentation
The template ships with 11 documentation files covering setup, customization, CMS schema references, and deployment. Most questions are answered in these guides. Please review the documentation before submitting a support request.

### Bug Fixes
If you encounter a bug in the template code as delivered — a broken layout, a JavaScript error, or a CSS issue that prevents the template from rendering correctly — it will be addressed. Bug fix support is provided for six months from the date of purchase.

### Setup Guidance
Reasonable questions about getting the template running, understanding the file structure, and following the documented customization steps are covered.

## What Is Not Included

- **Custom development.** Building new pages, adding features not present in the template, or integrating with third-party services falls outside support scope.
- **Content creation.** Writing copy, sourcing photography, or creating graphics for your clinic is not included.
- **Webflow migration.** The CMS schemas are documented for reference, but hands-on Webflow build-out is not part of support.
- **Hosting and deployment troubleshooting.** Guidance is provided in the documentation, but debugging server configurations or platform-specific issues is outside scope.
- **Third-party tools.** Integration with booking systems, payment processors, analytics platforms, or marketing tools is not covered.

## How to Get Help

Submit your question by email or through the messaging system on the platform where you purchased the template. Include the following:

1. A clear description of the issue or question
2. The page and section where the issue occurs (if applicable)
3. A screenshot if the issue is visual
4. Your browser and device information (if reporting a rendering problem)

## Response Times

- **Initial response:** Within 2 business days
- **Bug fix resolution:** Typically within 5 business days, depending on complexity
- **General questions:** Within 2 business days

Support is available Monday through Friday. Requests submitted on weekends or holidays will be addressed on the next business day.

## Support Period

Active support is provided for six months from the date of purchase. After this period, you retain full access to the template files and documentation, but new support requests will not be accepted.
