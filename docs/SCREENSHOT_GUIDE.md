## Screenshot Specifications

| Platform | Recommended Size | Format |
|----------|-----------------|--------|
| Gumroad | 1280x720px or 1600x900px | PNG |
| Lemon Squeezy | 1280x720px minimum | PNG |
| Webflow Marketplace | 1440x900px | PNG |

## Tips

- Use a clean browser window (no extensions, no bookmarks bar)
- Use incognito mode for clean screenshots
- Capture at 1440px viewport width for desktop shots
- Use 393px width for mobile shots (iPhone 14 Pro)
- Add subtle browser chrome or mockup frames for marketplace listings
- Consider creating a 30-second GIF showing scroll animations
