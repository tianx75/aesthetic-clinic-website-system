# Cover Image Brief

## Purpose

Create a marketplace cover image for the Aesthetic Clinic Website System product listing. The image should communicate premium quality, professional design, and the scope of the template at a glance.

## Dimensions

Prepare two versions:

- **Gumroad:** 1280 x 720 px
- **Lemon Squeezy:** 1600 x 900 px

Both should use the same composition, scaled to fit. Design at 1600 x 900 and export a cropped/resized version for Gumroad.

## Background

- Solid or subtly textured cream/ivory background (#FAF8F5 or similar from the template palette)
- Optional: very faint gold gradient accent in one corner to add depth without distraction

## Layout

### Primary Mockup (Center-Left)

Place a desktop browser mockup showing the homepage at a slight angle (5-10 degree perspective tilt). The screenshot should clearly show the hero section with its typography and layout.

### Secondary Mockup (Right, Overlapping)

Place a mobile device mockup showing either the treatments catalog page or the consultation page. Position it overlapping the bottom-right area of the desktop mockup to demonstrate responsiveness.

### Text Overlay (Top-Right or Bottom-Left)

Product name and key selling point:

```
Aesthetic Clinic Website System
16-Page Premium Template
```

Set the product name in Cormorant Garamond (or a similar serif) in charcoal (#2C2C2C). Set the subtitle in Montserrat in a lighter weight, slightly smaller. Keep text minimal — the mockups should do most of the communicating.

### Feature Badges (Bottom Edge)

A row of small, understated badges or tags:

- "16 Pages"
- "Fully Responsive"
- "No Dependencies"
- "CMS-Ready"

Style these as pill-shaped badges with a thin gold (#C9A96E) border on a cream background, using Montserrat at a small size. Do not make them visually dominant.

## Color Palette

Match the template's design language:

| Role | Color |
|---|---|
| Background | #FAF8F5 (cream) |
| Primary text | #2C2C2C (charcoal) |
| Accent | #C9A96E (gold) |
| Secondary background | #F5F0EB (ivory) |
| Light border | #E8E0D8 |

## Typography

- Headings: Cormorant Garamond, regular or semibold
- Body/badges: Montserrat, light or regular weight

## General Guidelines

- Keep the composition clean with generous whitespace
- Avoid promotional language beyond the product name and subtitle — no "Best Seller" or "Limited Time" badges
- The image should look like it belongs to the product it represents: calm, refined, premium
- Ensure all text is legible at thumbnail sizes (marketplace grid view)
- Export as PNG at high quality; provide a JPG fallback if file size is a concern

## Source Screenshots

Use the existing homepage and mobile screenshots from the project. If additional page screenshots are needed for variety, capture the treatments catalog or consultation page.
