### What's Inside

**16 pages. One system. Every touchpoint covered.**

- **Homepage** — Brand positioning with hero, philosophy, team, testimonials, and dual CTA
- **Treatment Catalog** — Filterable by category with real-time search across 18 treatments
- **Protocol System** — Three-phase treatment methodology (Prepare · Treat · Recover) with hub page, detail page, and individual protocol deep-dives
- **Technology Showcases** — Four device pages with specifications, clinical evidence sections, and modal overlays
- **Consultation Booking** — Trust-building form with visual timeline, social proof strip, and contact details
- **Academy** — Education and authority section with workshop tiers and credibility markers
- **Clinical Programs** — Multi-session structured treatment programs

### Design

Cormorant Garamond + Montserrat. Cream, ivory, charcoal, and gold. CSS custom properties you can change in one place. Scroll-triggered reveal animations. Responsive across desktop, tablet, and mobile.

This is not a design that needs to be "fixed." It is a finished editorial design system.

### Technical

- Pure HTML, CSS, and JavaScript — no React, no Tailwind, no build step
- Deploy on Netlify, Vercel, GitHub Pages, shared hosting, or any static server
- CSS custom properties for global theming
- IntersectionObserver-powered animations
- 7 CMS collection schemas documented (Treatments, Doctors, Technologies, Articles, Case Studies, Testimonials, FAQs) — ready for Webflow or any headless CMS
