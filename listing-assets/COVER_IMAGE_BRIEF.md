Create a single hero image for Gumroad and Lemon Squeezy product listings.

## Design Specifications

**Background:** Solid cream `#F8F4EE` or subtle gradient from `#F8F4EE` to `#F0EBE3`

**Title text:**
- "AESTHETIC CLINIC" — Cormorant Garamond, 48px, `#1A1816` (charcoal)
- "WEBSITE SYSTEM" — Cormorant Garamond, 48px, `#B8965A` (gold)

**Subtitle:**
- "Premium Medical Website Template" — Montserrat, 16px, `rgba(26,24,22,0.6)`

**Feature badges:**
- "16 Pages" · "Pure HTML/CSS/JS" · "Zero Dependencies"
- Montserrat 11px uppercase, `#B8965A` gold text or gold-bordered pills

**Mockups:**
- Desktop: Use `screenshots/homepage-hero.png` inside a minimal browser chrome frame
- Mobile: Use `screenshots/mobile-preview.png` inside an iPhone frame, overlapping the desktop slightly

**Do NOT include:**
- Price on the cover
- "Sale" or "discount" language
- Busy backgrounds or stock photos

## Tools to Create

- **Canva** — Use "Product Mockup" template, customize with brand colors
- **Figma** — Free browser mockup frames available in community
- **Placeit** — Automated device mockup generator
